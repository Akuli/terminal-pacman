# terminal-pacman
PacMan game clone that runs on terminal


## Running the game

```
git clone https://github.com/Akuli/terminal-pacman
cd terminal-pacman
python3 -m pacman
```

## Developing

In `terminal-pacman` folder, cloned as above:

```
python3 -m venv env
. env/bin/activate
pip install -r requirements-dev.txt
```

Run `./lint` to check and format code.
